# Raz Ecommerce - Backend

This project is a back-end ecommerce built using for Strapi Headless CMS.


## Install
```sh
npm install
```

## Start
Running it, will run the Strapi server, here http://localhost:1337.
```sh
npm run develop
```

the front-end ecommerce built using Next.js.
